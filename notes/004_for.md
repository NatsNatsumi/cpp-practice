# 004_for.cpp

## 目的
- C++の`for`文による繰り返し処理の基本を理解する
- `i++`による加算処理とループの仕組みを確認する

```cpp
//for文　出力は1,2,3

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 3; i++) {
        cout << i << endl;
    }
    return 0;
}
```

## 実行結果
1
2
3

## 学んだこと
- `for` 文を使って一定回数繰り返し処理ができる
- `i++` でループ変数を1ずつ増やせる
- `cout` でループごとに値を出力できる
