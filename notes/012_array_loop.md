# 012_array_loop.cpp

## 目的
- 配列と `for` 文を組み合わせて要素を順に処理する方法を学ぶ
- 繰り返し処理の実践的な使い方を理解する

```cpp
int scores[3] = {85, 90, 78};

for (int i = 0; i < 3; i++) {
    cout << scores[i] << endl;
}
```

## 実行結果
85
90
78

## 学んだこと
- `for` 文で配列の全要素を順番に処理できる
- 配列のインデックスは `0` から始まり、 `サイズ - 1` まで
- 配列サイズが決まっている場合、ループ条件に使える
